# 🧪 React Katas

Repositório de **treinos de front-end com React + JavaScript**. Cada pasta é um exercício pequeno e isolado, com foco em fundamentos: estado, renderização condicional, tratamento de dados ausentes e tomada de decisão na UI.

> Não é um projeto de produto — é meu caderno de prática. O projeto "de verdade" é o [Sistema de Controle Financeiro Pessoal](https://github.com/RafhaellPrates/Sistema-de-Controle-Financeiro-Pessoal).

## Por que existe

Venho do back-end (Node/Express) e estou construindo base sólida de front. Cada kata define **requisitos de negócio** e **critérios de aceite** antes de uma linha de código, pra eu treinar a pensar como engenheiro — não só "fazer funcionar".

## Katas

| # | Kata | Treina | Status |
|---|------|--------|--------|
| 01 | [medidor-senha](./medidor-senha) | estado ao vivo, renderização condicional, estado inicial vazio, decisão por faixas | 🚧 em andamento |

## Stack

- React + Vite
- JavaScript

## Como rodar

Cada kata é um projeto Vite independente. Dentro da pasta do kata:

```bash
npm install
npm run dev
```
